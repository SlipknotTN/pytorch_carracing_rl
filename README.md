# README

This is an implementation of Q-Learning to solve this Open AI
gym environment https://gym.openai.com/envs/CarRacing-v0/ using PyTorch.

### Requirements

Tested with:
- Ubuntu 18
- Nvidia RTX 2070 card
- Cuda 10.2
- CuDNN 7.6.5

```
pip install -r requirements.txt
```

### How to play the game

```
./play.sh
```
